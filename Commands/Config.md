# git config 

When first setting up git, you'll need to work with up the git config commandsin order to set up your identity.

For example, your identity can be set up with the commands such as:
```
git config --global user.name "Braulio Tejeda"
git config --global user.email brauliort218@gmail.com
```
For more information, view the [first time git setup](https://git-scm.com/book/en/v2/Getting-Started-First-Time-Git-Setup) documentation.
